# 本项目停止更新，请使用[VUE3+VITE 版本](https://github.com/chao921125/vue-vite)

# 对应分支说明

```
main = admin : 简易的admin管理端搭建
vw : 即将px转为vw
web : 仅仅布局做了调整
h5 : 适配手机端
```

# 目录结构

```
┌─package.json 包管理
├─public
│  ├─favicon.ico
│  ├─img
│  ├─index.html
│  └─robots.txt 反爬虫
├─src
│  ├─App.vue
│  ├─api 接口定义
│  ├─assets 静态资源
│  ├─components 公共组件
│  ├─libs 工具库
│  ├─main.js 启动入口
│  ├─mock 数据mock
│  ├─plugins 组件工具
│  ├─registerServiceWorker.js
│  ├─router 路由
│  ├─store vuex
│  └─views 页面
└─vue.config.js 启动配置
```
